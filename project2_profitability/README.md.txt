# Проект 2: Анализ прибыльности и динамика продаж

## Цель
Выявить топ-15 подкатегорий по выручке, оценить их рентабельность и определить убыточные зоны.

## Инструменты
- SQL (SQLite Studio) — извлечение и агрегация данных
- Python (Pandas, Matplotlib) — визуализация
- Power BI — интерактивный дашборд

## SQL-запрос
Оконные функции, CASE, группировки, ранжирование.

## Результаты
- Наибольшая выручка у категорий Phones, Chairs, Storage.
- Высокая рентабельность (>20%) у Accessories, Copiers, Paper.
- Убыточная подкатегория: Bookcases.

## Как воспроизвести
1. Открыть query.sql в SQLite Studio.
2. Сохранить результат в CSV.
3. Загрузить CSV в Python (графики) и Power BI (дашборд).
